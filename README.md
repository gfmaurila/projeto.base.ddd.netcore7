# Estrutura da API
- ASP.NET Core 7.0: Framework para desenvolvimento da Microsoft.
- DDD: Domain Drive Design modelagem de desenvolvimento orientado a injeção de dependência.
- AutoMapper: Biblioteca para realizar mapeamento entre objetos.
- Entity Framework Core 7.0
- SQL Server
- Docker & Docker Compose

# Executando a aplicação
- Add-Migration Inicial -Context SQLServerContext
- Update-Database -Context SQLServerContext
- docker-compose up --build
